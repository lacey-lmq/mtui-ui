# Box 容器
盒子组件

### 示例

<mt-box bg="lightblue" border="true">山有木兮木有枝，心悦君兮君不知。</mt-box>


### 代码
```html
<mt-box border="true" center="true" padding="30px" >
    <img src="/star.png" style="width: 60px; height: 60px">
</mt-box>
```


### Attributes
| 参数      | 说明 | 类型 | 是否必要 | 默认值 |
| ----------- | ----------- | ----------- | ----------- | ----------- |
| border      | 盒子的边框       | Boolean | false | false |
